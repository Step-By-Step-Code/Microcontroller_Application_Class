# GPIO_Controll
## DDRx [DDRB, DDRC, DDRD]
- **1 → 출력, 0 → 입력**
## PORTx
- 출력일 때 → 해당 핀의 출력값을 결정
    - 1 → HIGH (Vcc 출력)
    - 0 → LOW (GND 출력)
- 입력일 때 → 풀업 저항 제어
    - 1 → 내부 풀업 ON (기본 HIGH 상태 유지) → **평소에 1로 읽힘**
    - 0 → 풀업 OFF (High-Z, 외부 회로에서 제어 필요)
## PINx
- 실제 핀의 현재 전기적 상태를 읽는 레지스터 (입력값 확인 용도)
- 읽기만 가능

## 비트 연산 종류
1. **특정 비트만 1로 세팅, 나머지는 유지**
    - DDRD |= (1 << PD2);
    - DDRD |= 0x35;

2. **특정 비트만 0으로 클리어**
    - DDRD &= ~(1 << PD2);
    - DDRD &= ~(0x35);

3. 원하는 위치만 바꾸는 비트연산
    ```c
    uint8_t mask   = 0b00111110;   // 바꾸고 싶은 중간 5자리 (비트1~5)
    uint8_t newVal = 0b00010100;   // 원하는 값 (비트1~5 영역에 맞춰 준비)
    DDRD = (DDRD & ~mask) | (newVal & mask);
    ```
    - 동작 원리
        - (DDRD & ~mask) → 중간 5자리만 0으로 클리어
        - (newVal & mask) → 새로운 값에서 필요한 부분만 추출

4. **특정 비트 값만 0 또는 1로 가져오기**
    - uint8_t bit_val = (DDRD >> PD2) & 0x01;

5. **하위 4비트만 추출**
    - DDRD & 0b00001111;

6. 특정 비트만 토글
    - DDRD ^= (1 << PD2);
